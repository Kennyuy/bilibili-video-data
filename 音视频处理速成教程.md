```bash
# 改变分辨率

# 改变码率
```

## ⚡ Day 3: 去重核心 - 时间戳操作

### 3.1 setpts 原理

视频由一系列帧组成，每帧有一个时间戳（PTS = Presentation Time Stamp）。
**修改 PTS = 改变播放速度 = 绕过平台去重算法**

### 3.2 基础变速

```bash
# 2 倍速（时间压缩）
ffmpeg -i input.mp4 -filter:v "setpts=0.5*PTS" output.mp4

# 0.5 倍速（慢放）
ffmpeg -i input.mp4 -filter:v "setpts=2.0*PTS" output.mp4

# 音频同步变速
ffmpeg -i input.mp4 -filter:v "setpts=0.5*PTS" -filter:a "atempo=2.0" output.mp4
```

### 3.3 非线性变速（核心去重技巧）

```bash
# 前快后慢（前 5 秒 1.5 倍，之后 0.8 倍）
ffmpeg -i input.mp4 -filter:v "setpts=if(lt(t,5),0.67*PTS,1.25*PTS)" output.mp4

# 正弦波变速（呼吸效果）
ffmpeg -i input.mp4 -filter:v "setpts=(1+0.1*sin(t))*PTS" output.mp4

# 随机变速（每 3 秒变化一次）
ffmpeg -i input.mp4 -filter:v "setpts=(0.9+0.2*random(t/3))*PTS" output.mp4
```

### 3.4 MoviePy 实现

```python
from moviepy.editor import *
import numpy as np

def nonlinear_speed(clip, speed_function):
    """
    speed_function: t -> speed 的函数
    """
    # 创建时间映射
    duration = clip.duration
    frames = []
    
    current_time = 0
    dt = 1/clip.fps
    
    while current_time < duration:
        speed = speed_function(current_time)
        frame = clip.get_frame(current_time)
        frames.append(frame)
        current_time += dt * speed
    
    # 重新组装（简化版，生产环境用更高效的实现）
    return clip.speedx(np.mean([speed_function(t) for t in np.linspace(0, duration, 100)]))

# 使用示例
clip = VideoFileClip("input.mp4")
# 前慢后快
clip = nonlinear_speed(clip, lambda t: 0.8 if t < 5 else 1.2)
```

## 🎵 Day 4: 音频处理与相位调整

### 4.1 音频去重原理

平台通过**音频指纹**识别重复内容。绕过方法：
- 改变音调（pitch）
- 改变相位（phase）
- 添加背景音
- 时间拉伸

### 4.2 音调调整

```bash
# 升高音调（不改变时长）
ffmpeg -i input.mp4 -filter:a "asetrate=44100*1.1,atempo=1.1" output.mp4

# 降低音调
ffmpeg -i input.mp4 -filter:a "asetrate=44100*0.9,atempo=0.9" output.mp4

# 微调（推荐去重用）
ffmpeg -i input.mp4 -filter:a "asetrate=44100*1.03,atempo=1.03" output.mp4
```

### 4.3 相位调整

```bash
# 左右声道交换
ffmpeg -i input.mp4 -filter:a "pan=stereo|c0=c1|c1=c0" output.mp4

# 添加延迟（制造回声）
ffmpeg -i input.mp4 -filter:a "adelay=150|150" output.mp4

# 添加混响
ffmpeg -i input.mp4 -filter:a "aecho=0.8:0.9:1000:0.3" output.mp4
```

### 4.4 音频混合

```bash
# 添加背景音乐（原声 80% + BGM 20%）
ffmpeg -i video.mp4 -i bgm.mp3 \
  -filter_complex "[0:a]volume=0.8[a];[1:a]volume=0.2[b];[a][b]amix=inputs=2:duration=first" \
  -c:v copy output.mp4
```

### 6.1 自动字幕（Whisper）

```python
import whisper

# 加载模型（tiny/base/small/medium/large）
model = whisper.load_model("base")

# 转录
result = model.transcribe("audio.mp3", language="zh")

# 生成 SRT
def write_srt(segments, filename):
    with open(filename, "w", encoding="utf-8") as f:
        for i, seg in enumerate(segments):
            start = format_time(seg["start"])
            end = format_time(seg["end"])
            text = seg["text"].strip()
            
            f.write(f"{i+1}\n")
            f.write(f"{start} --> {end}\n")
            f.write(f"{text}\n\n")

def format_time(seconds):
    hours = int(seconds // 3600)
    minutes = int((seconds % 3600) // 60)
    secs = int(seconds % 60)
    millis = int((seconds % 1) * 1000)
    return f"{hours:02d}:{minutes:02d}:{secs:02d},{millis:03d}"

write_srt(result["segments"], "subtitle.srt")
```

```python
from moviepy.editor import *

# 创建字幕片段
def create_subtitle_clip(text, duration, position='bottom'):
    txt = TextClip(
        text, 
        fontsize=24, 
        color='white', 
        font='Arial',
        bg_color='transparent',
        stroke_color='black',
        stroke_width=1
    )
    return txt.set_duration(duration).set_pos(position)

# 加载 SRT 并创建字幕轨道
def add_subtitles(video, srt_file):
    subtitles = parse_srt(srt_file)
    subtitle_clips = []
    
    for sub in subtitles:
        clip = create_subtitle_clip(sub['text'], sub['end'] - sub['start'])
        clip = clip.set_start(sub['start'])
        subtitle_clips.append(clip)
    
    return CompositeVideoClip([video] + subtitle_clips)
```

## 🚀 Day 7: 综合项目 - 完整去重流水线

### 7.1 完整脚本

```python
#!/usr/bin/env python3
"""
短视频去重处理流水线
功能：自动分析 + 多层去重 + 字幕生成 + 批量处理
"""

import os
import sys
import random
import subprocess
import whisper
from moviepy.editor import *
from moviepy import vfx
from datetime import datetime

class VideoDeduplicator:
    def __init__(self, config=None):
        self.config = config or self.default_config()
        
    def default_config(self):
        return {
            'scale_range': (0.98, 1.02),      # 缩放范围
            'speed_range': (0.95, 1.05),       # 速度范围
            'brightness_range': (-0.03, 0.03), # 亮度范围
            'contrast_range': (0.97, 1.03),    # 对比度范围
            'saturation_range': (0.95, 1.05),  # 饱和度范围
            'pitch_shift': 1.02,               # 音调偏移
            'add_subtitles': True,             # 自动字幕
            'add_bgm': False,                  # 添加背景音乐
        }
    
    def generate_random_params(self):
        """生成随机去重参数"""
        return {
            'scale': random.uniform(*self.config['scale_range']),
            'speed': random.uniform(*self.config['speed_range']),
            'brightness': random.uniform(*self.config['brightness_range']),
            'contrast': random.uniform(*self.config['contrast_range']),
            'saturation': random.uniform(*self.config['saturation_range']),
        }
    
    def process_video(self, input_path, output_path):
        """处理单个视频"""
        print(f"🎬 开始处理：{input_path}")
        
        # 1. 加载视频
        clip = VideoFileClip(input_path)
        params = self.generate_random_params()
        
        # 2. 变速处理
        clip = clip.speedx(params['speed'])
        
        # 3. 视觉去重
        clip = clip.resize(params['scale'])
        clip = clip.fx(vfx.ColorGamma, gamma=1.0 + random.uniform(-0.05, 0.05))
        clip = clip.fx(vfx.MultiplyColor, 1.0 + params['brightness'])
        
        # 4. 音频去重
        if clip.audio is not None:
            # 提取音频并用 FFmpeg 处理
            temp_audio = f"/tmp/audio_{datetime.now().timestamp()}.mp3"
            clip.audio.write_audiofile(temp_audio)
            
            # 音调调整
            processed_audio = f"/tmp/audio_processed_{datetime.now().timestamp()}.mp3"
            subprocess.run([
                'ffmpeg', '-i', temp_audio,
                '-filter:a', f"asetrate=44100*{self.config['pitch_shift']},atempo={self.config['pitch_shift']}",
                '-y', processed_audio
            ])
            
            # 替换音频
            clip = clip.set_audio(AudioFileClip(processed_audio))
            
            # 清理临时文件
            os.remove(temp_audio)
            os.remove(processed_audio)
        
        # 5. 自动字幕
        if self.config['add_subtitles']:
            clip = self.add_auto_subtitles(clip)
        
        # 6. 输出
        clip.write_videofile(
            output_path,
            codec='libx264',
            audio_codec='aac',
            fps=30,
            preset='medium',
            bitrate='2000k'
        )
        
        print(f"✅ 处理完成：{output_path}")
        return output_path
    
    def add_auto_subtitles(self, clip):
        """添加自动字幕"""
        # 提取音频
        temp_audio = f"/tmp/sub_{datetime.now().timestamp()}.mp3"
        clip.audio.write_audiofile(temp_audio)
        
        # Whisper 转录
        model = whisper.load_model("base")
        result = model.transcribe(temp_audio, language="zh")
        
        # 创建字幕轨道
        subtitle_clips = []
        for seg in result["segments"]:
            txt = TextClip(
                seg["text"].strip(),
                fontsize=24,
                color='white',
                font='Arial',
                stroke_color='black',
                stroke_width=1
            )
            txt = txt.set_duration(seg["end"] - seg["start'])
            txt = txt.set_start(seg["start"])
            txt = txt.set_pos(('center', 0.85), relative=True)
            subtitle_clips.append(txt)
        
        # 清理临时文件
        os.remove(temp_audio)
        
        # 合成
        return CompositeVideoClip([clip] + subtitle_clips)
    
    def batch_process(self, input_dir, output_dir):
        """批量处理"""
        os.makedirs(output_dir, exist_ok=True)
        
        video_files = [f for f in os.listdir(input_dir) if f.endswith(('.mp4', '.mov', '.avi'))]
        
        for i, filename in enumerate(video_files, 1):
            input_path = os.path.join(input_dir, filename)
            output_filename = f"processed_{i:03d}_{filename}"
            output_path = os.path.join(output_dir, output_filename)
            
            try:
                self.process_video(input_path, output_path)
            except Exception as e:
                print(f"❌ 处理失败 {filename}: {e}")

# 使用示例
if __name__ == "__main__":
    dedup = VideoDeduplicator()
    
    # 单个处理
    dedup.process_video("input.mp4", "output.mp4")
    
    # 批量处理
    # dedup.batch_process("./input_videos", "./output_videos")
```

### 7.2 FFmpeg 一键命令版

```bash
#!/bin/bash
# dedup.sh - 一键去重脚本

INPUT=$1
OUTPUT=$2

# 随机参数
SCALE=$(echo "scale=4; 0.98 + $(od -An -N2 -tu2 /dev/urandom | tr -d ' ') / 65535 * 0.04" | bc)
BRIGHT=$(echo "scale=4; -0.03 + $(od -An -N2 -tu2 /dev/urandom | tr -d ' ') / 65535 * 0.06" | bc)
CONTRAST=$(echo "scale=4; 0.97 + $(od -An -N2 -tu2 /dev/urandom | tr -d ' ') / 65535 * 0.06" | bc)
SPEED=$(echo "scale=4; 0.95 + $(od -An -N2 -tu2 /dev/urandom | tr -d ' ') / 65535 * 0.10" | bc)

echo "🎬 去重参数:"
echo "  缩放：$SCALE"
echo "  亮度：$BRIGHT"
echo "  对比度：$CONTRAST"
echo "  速度：$SPEED"

ffmpeg -i "$INPUT" \
  -filter:v "scale=iw*$SCALE:ih*$SCALE,eq=brightness=$BRIGHT:contrast=$CONTRAST" \
  -filter:a "asetrate=44100*1.02,atempo=1.02" \
  -c:v libx264 -c:a aac \
  -preset medium -crf 23 \
  "$OUTPUT"

echo "✅ 完成：$OUTPUT"
```

## 📚 附录：专业人员经验总结

### A. 去重效果检测

```bash
# 计算视频 MD5（基础检测）
md5sum video.mp4

# 提取关键帧对比
ffmpeg -i video.mp4 -vf "select=eq(n\,0)" -vframes 1 frame1.png
ffmpeg -i video2.mp4 -vf "select=eq(n\,0)" -vframes 1 frame2.png
compare -metric AE frame1.png frame2.png diff.png
```

### B. 平台去重机制分析

| 平台 | 检测维度 | 绕过策略 |
|------|----------|----------|
| 抖音 | 视频指纹 + 音频指纹 | 变速 + 缩放 + 音调调整 |
| 快手 | 关键帧哈希 | 删帧 + 添加滤镜 |
| B 站 | MD5 + 元数据 | 重编码 + 清除元数据 |
| YouTube | Content ID | 多层去重叠加 |

### C. 推荐参数组合

**轻度去重（保留画质）：**
- 缩放：1.01-1.02
- 速度：0.98-1.02
- 亮度：±0.02
- 音调：1.01

**中度去重（平衡质量）：**
- 缩放：1.02-1.05
- 速度：0.95-1.05
- 亮度：±0.03
- 对比度：0.97-1.03
- 音调：1.02

**重度去重（最大安全）：**
- 缩放：0.98-1.05
- 速度：0.90-1.10（非线性）
- 多层滤镜叠加
- 音频混音 + 音调调整
- 删帧/插帧

### D. 常见错误与解决

| 错误 | 原因 | 解决 |
|------|------|------|
| 音画不同步 | 变速未同步音频 | 同时使用 setpts + atempo |
| 画质下降 | 重编码参数过低 | CRF 18-23，preset medium |
| 字幕乱码 | 编码问题 | 添加 `-sub_charenc UTF-8` |
| 处理太慢 | 滤镜过多 | 用 GPU 加速：`-hwaccel cuda` |
